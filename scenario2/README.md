# Scenario 2

## Goal

Practice debugging and improving an existing API.

## Existing functionality

The application currently supports:
- creating a newsletter subscriber,
- listing all subscribers.

## Reported issue

The product team says duplicate subscriptions sometimes happen when the same email address is submitted with different casing (for example `ANNA@example.com` and `anna@example.com`).

## Task

Investigate the issue and improve the implementation so that email addresses are treated consistently.
