# Practice Live Coding Repository

This repository contains 2 practice scenarios for the live coding session.

Common stack for both scenarios:
- Java 21
- Kotlin 1.9.25
- Spring Boot 3.3.5
- Gradle
- H2 in-memory database

## Goals

These scenarios are intended to help you verify your local setup and practice working in a small Kotlin/Spring Boot project.

They are similar in style to the live session, but they do **not** mirror the exact interview tasks.

## How to use

1. Open one scenario at a time in your IDE.
2. Run the test suite.
3. Read the scenario README.
4. Implement the requested changes until the tests pass or until you are satisfied with your solution.

## Notes

- Please use Java 21.
- If Gradle test execution fails because of JUnit Platform resolution, refresh Gradle and make sure the wrapper and dependencies are picked up correctly.
